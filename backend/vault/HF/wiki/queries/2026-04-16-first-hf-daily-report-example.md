# First HF Daily Report Example

## Summary

This example shows what a filled HF daily report looks like using the standard template.

## Example

```md
# HF Daily Report - 2026-04-16

## Snapshot

- Day status: Yellow
- Sleep hours: 5.5
- Sleep quality (1-5): 2
- Sleep timing: very late
- Morning energy (1-5): 2
- Midday energy (1-5): 3
- Evening energy (1-5): 2
- Mental clarity (1-5): 2
- Focus block done: no
- Mood (1-5): 3
- Stress (1-5): 4
- Steps or walk: 20-minute walk
- Training: none
- Mobility: no
- Protein quality: mixed
- Hydration: adequate
- Meal control: mixed
- Pain or discomfort: mild
- Illness signal: none

## Interpretation

This looks like a recovery-constrained day driven mainly by poor sleep timing and low-quality rest. Clarity and energy were both weak, and stress was high enough that adding heavy demands would probably create a second bad day rather than a rebound.

## Wins

- Managed at least some movement with a short walk
- Hydration did not completely collapse

## Risks

- A second late night could push this into a red pattern
- Missed focus work suggests cognitive performance is already impaired

## Corrective Action For Tomorrow

- Sleep before 12:00 AM and do a 15-minute mobility session before noon

## Notes

- No major illness signal, so this appears to be routine failure rather than acute sickness
```

## Why This Example Matters

- It demonstrates the expected brevity.
- It shows how numbers translate into interpretation.
- It gives HF a concrete starting standard for future reporting.
