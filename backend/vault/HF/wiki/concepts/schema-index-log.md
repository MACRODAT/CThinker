# Schema Index Log

## Summary

Three control artifacts keep the vault disciplined: the schema (`Agents.md`), the index (`index.md`), and the chronological log (`log.md`).

## Details

- `Agents.md` defines rules, workflows, naming conventions, and the division between raw sources and the maintained wiki.
- `index.md` is the main catalog. It tells the agent what pages exist and where to look next.
- `log.md` records the timeline of ingests, queries, lint passes, and structural changes.

## Importance

These files let a general-purpose coding agent behave like a reliable wiki maintainer instead of a stateless chatbot.
